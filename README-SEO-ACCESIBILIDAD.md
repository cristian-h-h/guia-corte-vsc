# Mejoras de SEO y Accesibilidad para GuiaDeCorte.cl

Este documento describe las mejoras implementadas para optimizar el SEO y la accesibilidad del sitio web GuiaDeCorte.cl.

## Mejoras de SEO

### 1. Componente SEO Mejorado
- Metaetiquetas completas para SEO
- Soporte para Open Graph y Twitter Cards
- Configuración de idioma y región
- Soporte para múltiples locales
- Metadatos para artículos y productos

### 2. Datos Estructurados (Schema.org)
- Esquema de organización
- Esquema de sitio web
- Esquema de producto
- Esquema de artículo de blog
- Esquema de página de contacto
- Esquema de preguntas frecuentes (FAQ)
- Esquema de migas de pan (breadcrumbs)
- Esquema de video
- Esquema de galería de imágenes

### 3. Sitemap.xml Mejorado
- Soporte para imágenes
- Soporte para videos
- Prioridades y frecuencias de cambio
- Fechas de última modificación

### 4. Robots.txt Optimizado
- Reglas específicas para diferentes bots
- Bloqueo de directorios sensibles
- Configuración de Crawl-delay
- Referencia al sitemap

### 5. Optimización de Imágenes
- Carga perezosa (lazy loading)
- Placeholders durante la carga
- Manejo de errores

- Soporte para prioridad de carga

### 6. Enlaces Optimizados
- Atributos rel para enlaces externos
- Títulos descriptivos
- Manejo seguro de enlaces externos

### 7. Manifiesto Web
- Configuración para PWA
- Iconos en múltiples tamaños
- Colores de tema
- Accesos directos
- Capturas de pantalla

## Mejoras de Accesibilidad

### 1. Panel de Accesibilidad
- Alto contraste

- Fuente para dislexia

### 2. Navegación por Teclado
- Enlaces de salto (skip links)
- Enfoque visible mejorado
- Orden de tabulación lógico

### 3. Compatibilidad con Lectores de Pantalla
- Textos alternativos para imágenes
- Roles ARIA
- Etiquetas descriptivas
- Anuncios de estado

### 4. Estilos de Accesibilidad
- Modo de alto contraste
- Tamaño de texto ajustable
- Reducción de animaciones
- Fuente OpenDyslexic

### 5. Estructura Semántica
- Uso de elementos HTML5 semánticos
- Jerarquía de encabezados correcta
- Etiquetas figure y figcaption
- Roles ARIA para componentes personalizados

## Cómo Usar las Funciones de Accesibilidad

1. **Panel de Accesibilidad**: Haz clic en el botón flotante de accesibilidad en la esquina inferior derecha para abrir el panel de opciones.

2. **Alto Contraste**: Activa esta opción para mejorar la legibilidad con colores de alto contraste.

3. **Texto Grande**: Aumenta el tamaño del texto en todo el sitio para facilitar la lectura.

4. **Reducir Movimiento**: Desactiva animaciones y transiciones para reducir distracciones.

5. **Fuente para Dislexia**: Cambia a una fuente diseñada específicamente para personas con dislexia.

## Pruebas de Accesibilidad

Se ha configurado un archivo `.a11yrc` para realizar pruebas automatizadas de accesibilidad según los estándares WCAG 2.1 AA. Para ejecutar las pruebas, instala las dependencias necesarias y ejecuta el comando de prueba de accesibilidad.
